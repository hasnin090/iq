## 📋 المسارات المتاحة:
- `/api/auth/login` - تسجيل الدخول
- `/api/auth/logout` - تسجيل الخروج  
- `/api/auth/session` - التحقق من الجلسة
- `/api/users` - إدارة المستخدمين
- `/api/projects` - إدارة المشاريع
- `/api/transactions` - إدارة المعاملات
- وجميع المسارات الأخرى المعرفة في `routes.ts`

## 🚀 الخطوات التالية للنشر:

### للتطوير المحلي:
```bash
npm run dev
```
الخادم سيعمل على: http://localhost:3000

### للإنتاج:
```bash
npm run build
npm start
```

## 📁 الملفات المهمة:
